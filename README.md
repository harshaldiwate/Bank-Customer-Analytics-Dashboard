## Overview

This project analyzes customer, account, and transaction data using SQL and Power BI. The dashboard provides insights into customer demographics, account performance, loan distribution, regional balances, and transaction trends.

## Dataset

- Customers: 150 Records
- Accounts: 250 Records
- Transactions: 5,000+ Records

## SQL Concepts Used

- JOIN
- GROUP BY
- Aggregate Functions
- Filtering
- Sorting
- Primary Keys
- Foreign Keys

## Dashboard Features

- Total Customers
- Closed Accounts
- Loan Accounts
- High Loan Accounts
- Customer Distribution by Region
- Account Type Analysis
- Account Status Analysis
- Regional Balance Analysis
- Monthly Transaction Trends
- Top Customers by Balance

- SQL
